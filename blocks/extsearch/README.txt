Description
------------

This block is used to allow teachers to provide their students with an easy
way of using an external search engine from within a course page.

It currently supports these search providers:
- DigitalNZ (http://www.digitalnz.org)
- EDNA/Education Network Australia (http://www.edna.edu.au)
- Google Web Search (http://www.google.com)


Requirements
-------------

* Moodle 1.9
* PHP 5.2 or later


Installation
-------------

1- Unpack the module into your moodle install in order to create a
   blocks/extsearch directory

2- Visit the /admin/index.php page to trigger the module installation.

3- Change the default options in the block configuration.
